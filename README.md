# react-template-spa

This project is a template for creating a SPA in React.

## How to Use

execution

```
npm run dev
```

```
npm run test
```

```
npm run test-storybook
```

## Libraries

### Project Configuration

- husky
- lintstaged

- eslint
- prettier
- commitlint
- vite

### UI

- Mantine
- CSS Modules
- Recharts

### Testing

- Storybook
- Test Runner
- Jest
- MSW

### State Management

- Tanstack Query
- jotai
